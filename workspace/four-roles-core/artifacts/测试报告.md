# 测试报告

由**测试总监**在单次运行或某轮测试结束时，汇总当次的测试范围、用例、结果与发布建议。可与「测试记录」配合：记录表按条追加，本报告按 run 或版本汇总。

---

## 报告索引

| runId / 版本 | 报告时间 | 测试范围 | 结论（通过/不通过/建议） |
|--------------|----------|----------|--------------------------|
| （示例）run_xxx | 2026-02-24T11:00:00Z | 功能 X、Y | 建议发布 / 不通过，见下方 |

---

*由 Agent（测试总监角色）在 test 阶段完成或 release 阶段时，视需要追加/更新上表并填写当次报告正文。*
